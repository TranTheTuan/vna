- Verify health endpoints appear in Swagger UI
- Annotations must include correct tags (`health`), paths (`/healthz`, `/readyz`), and responses

## Implementation Steps

1. Run Swagger generation:

```bash
make docs
```

2. Verify the generated files include health endpoints:

```bash
grep -A5 "healthz" internal/docs/swagger.json
grep -A5 "readyz" internal/docs/swagger.json
```

3. Optionally start server and check Swagger UI:

```bash
make run
# Visit http://localhost:8080/swagger/index.html
```
